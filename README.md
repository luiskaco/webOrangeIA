# Orange Latam WordPress Theme

Tema de WordPress premium, auto-instalable y auto-configurable desarrollado para **Orange Latam**. Optimizado para velocidad y rendimiento utilizando HTML5 semántico, JavaScript nativo (Vanilla JS) y CSS puro.

## 🚀 Características Principales

- **Auto-configurable**: Al activar el tema, se crean automáticamente la página de inicio estática, los menús de navegación correspondientes y la configuración básica de WordPress.
- **Rendimiento Premium**: Sin dependencias ni frameworks CSS pesados. Construido con Vanilla JS y CSS nativo para garantizar tiempos de carga ultrarrápidos.
- **Interactividad Visual**:
  - Hero Slider interactivo.
  - Animaciones de revelado al hacer scroll.
  - Interruptor interactivo de servicios (Corporativo vs. Asuntos Sensibles).
  - Carrusel de premios y sectores.
- **Mobile-First**: Diseño responsivo y fluido adaptado a todos los dispositivos.

## 📂 Estructura del Repositorio

El repositorio está configurado para subir únicamente los componentes esenciales del desarrollo:
- `wp-content/themes/orange-latam/`: Archivos del tema de WordPress.
- `docs/`: Documentación del proyecto (PRD, Arquitectura, Modelo de Datos, etc.).
- `.agents/` y `.claude/`: Configuraciones de agentes de desarrollo e Inteligencia Artificial.

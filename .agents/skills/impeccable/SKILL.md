# Impeccable

Comandos disponibles via `/impeccable <comando>`:

| Comando | Uso |
|---------|-----|
| `audit` | Revisión técnica: a11y, performance, responsive |
| `critique` | Revisión UX: jerarquía, claridad, resonancia |
| `polish` | Pase final antes de ship |
| `harden` | Errores, i18n, overflow, edge cases |
| `bolder` | Amplificar diseños aburridos |
| `quieter` | Tonificar diseños sobrecargados |
| `typeset` | Corregir tipografía |
| `animate` | Agregar motion con propósito |
| `distill` | Simplificar al mínimo esencial |
| `colorize` | Introducir color estratégico |
| `layout` | Corregir espaciado y ritmo visual |

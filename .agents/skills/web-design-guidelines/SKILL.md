# Web Interface Guidelines

Antes de cada revisión, fetch las guidelines actualizadas desde:
```
https://raw.githubusercontent.com/vercel-labs/web-interface-guidelines/main/command.md
```

Luego revisar los archivos indicados contra todas las reglas del documento.
Reportar hallazgos en formato `archivo:línea — descripción del issue`.
